Overview

This is my personal website. I have been a professional frontend developer for 7+ years. I'm using this to house various projects and other things I'm interested in. Nothing here has been written with an LLM. This is all organic slop.

Technical Bits

This site is built as a static site using Vite and Handlebars. I use Lit to make web components like the app window and carousel. The games are mostly built with Godot and embedded using iframes. You can embed these games in your own website with an iframe. When possible a page will have an "embed" button that writes an html snippet to your clipboard.
